# Login Checklist

## Positive Scenarios 

- Verify login with valid credentials (standard_user)
- Verify successful redirection to Products page after login 

## Negative Scenarios 

- Verify login with invalid username 
- Verify login with invalid password 
- Verify login with invalid username and password 
- Verify login with empty username field 
- Verify login with empty password field 
- Verify login with empty username and password fields 
- Verify login with locked_out_user shows erorr message

## UI Verification 

- Verify Username field is displayed 
- Verify Password field is displayed 
- Verify Login button is displayed 
- Verify Login button is clickable 
- Verify error message is visible and readable 

## Validation 

- Verify password field masks entered characters 
- Verify fields accept user input 
- Verify error message remains consistent after multiple failed login attempts
