# Checkout Checklist

## User Information

- Verify First Name field accepts input
- Verify Last Name field accepts input
- Verify Postal Code field accepts input

## Validation

- Verify checkout with empty First Name
- Verify checkout with empty Last Name
- Verify checkout with empty Postal Code
- Verify error messages are displayed

## Order Completion

- Verify selected products are displayed in checkout overview
- Verify product prices are displayed correctly in checkout overview
- Verify total price is calculated correctly
- Verify successful order placement
- Verify confirmation page is displayed
- Verify Cancel button works
- Verify Finish button works
- Verify Back Home button works
